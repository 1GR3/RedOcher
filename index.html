Red Ocher

Named after one of the oldest pigments known to mankind, Red Ocher is a MacOS menu bar application (a.k.a. MenuBarExtra) that allows users to easily capture a portion of the screen, perform optical character recognition (OCR) on it using the Tesseract OCR library and return the text in the clipboard.

Features

  • Simple to use - stays out of the way.
  • Gets triggered on a custom shortcut.
  • Choose the specific language.
  • Define whitelist and blacklist for the characters.
