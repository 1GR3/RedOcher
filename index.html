Red Ocher

Red Ocher is a macOS menu bar application (a.k.a. MenuBarExtra) that allows users to capture a portion of the screen and perform optical character recognition (OCR) on the captured image using the Tesseract OCR library.

Features

  • Simple to use - stays out of the way.
  • Gets triggered on a custom shortcut.
  • Choose the specific language.
  • Define whitelist and blacklist for the characters.

Prerequisites

Before using this application, ensure that you have the following dependencies installed:

  • Xcode
  • Tesseract OCR library

Installation

  1. Clone the repository to your local machine.
  2. Open the project in Xcode.
  3. Build and run the project in Xcode to launch the application.

Usage

  1. Launch the Red Ocher application.
  2. Select the desired portion of the screen by clicking and dragging the mouse.
  3. The application will capture the selected portion of the screen.
  4. The captured image will be processed using Tesseract OCR.
  5. The recognized text will be displayed within the application's interface.
  6. Optionally, you can copy the recognized text to the clipboard for further use.
