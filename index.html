Named after one of the oldest pigments known to mankind, Red Ocher /ɹɛd ˈoʊkər/ is a macOS menu bar application (a.k.a. MenuBarExtra) that allows users to easily capture a portion of the screen, perform optical character recognition (OCR) on it using the Tesseract OCR library and return the text into the clipboard.

Weather you need to digitize an old document, copy a "protected" text from a web or qiuckly grab words out of a photo, Red Ocher is a simple and powerful solution.

Simple to Use

Sits in the menu bar, day and night...
...waiting to be triggered...
...and paste the text.

Powerful Features

Custom Shortcut

Choose 1 of 3 convenient presets or create a custom keyboard shortcut

Select language(s)

Choose one or more specific language from over 120 different language and script models.

Character lists

Specify the correct characters by creating a whitelist or/and a blacklist

To properly function,
Red Ocher requires
specific system settings.

To load Red Ocher on login (optional):
System Preferences > Users & Groups > [your username] > Login Items > add Red Ocher

To set a global keyboard shortcut:
System Preferences > Security & Privacy Privacy > Accessibility > add Red Ocher

To take a screenshot:
System Preferences > Security & Privacy Privacy > Screen Recording > add Red Ocher
