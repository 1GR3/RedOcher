Named after one of the oldest pigments known to mankind, Red Ocher is a macOS menu bar application (a.k.a. MenuBarExtra) that allows users to easily capture a portion of the screen, perform optical character recognition (OCR) on it using the Tesseract OCR library and return the text in the clipboard.

Weather you need to digitize an old document, copy the "protected" text from a web or qiuckly grab words out of a photo, Red Ocher is a simple and powerful solution.

Features

Custom Shortcut

Choose 1 of 3 convenient presets or create a custom keyboard shortcut

Select language(s)

Choose one or more specific language from over 120 different language and script models.

Create lists

Specify the correct characters by creating a whitelist or/and a blacklist

Usage

Sits in the menu bar, day and night...
...waiting for you to trigger it...
...and paste the text.
